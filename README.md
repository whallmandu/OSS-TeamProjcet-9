# Commit Message Convention

*[Type] branch-name(*optional modified-file): Title (imperative form)*

**Optional body (describe what and why you changed)*

**Optional (mandatory if related): Issue number and related details*

| Type | Description |
| --- | --- |
| **Feat** | Add a new feature |
| **Build** | Build-related changes (e.g., creating an executable file) |
| **Bugfix** | Fix a bug |
| **Refactor** | Improve code (performance, maintainability, readability) |
| **Docs** | Documentation-related changes |
| **Test** | Add or update test code |
| **Revert** | Revert previous code changes |

---

## Examples

```
[Feat] save&load: implement load function

Related to issue #123

```

```
[Bugfix] choice(choice.c): fix HP decrease mismatch

The text stated HP decreases by 5, but it actually decreased by 10.
Updated the text to correctly show 10.

```

```
[Test] develop: add debugging code

When entering "develop1" as the first input, all choices succeed.
When entering "develop2", all choices fail.

Issue #273

```

```
[Docs] main(README.md): update game rules

Added missing game rule descriptions to README.md.
Also clarified previously ambiguous explanations.

Resolved issue #347

```
